Meus Projetos

Projeto 1

Desenvolvi o front-end de uma loja virtual responsiva e moderna, com foco na experiência do usuário e na navegação intuitiva. O projeto foi construído utilizando tecnologias como HTML5, CSS3 e JavaScript, garantindo compatibilidade com dispositivos móveis e navegadores modernos. O design conta com uma página inicial atraente, listagem de produtos com filtros dinâmicos, página de detalhes de produto com informações. O objetivo foi criar uma interface amigável que facilite a navegação e a compra de produtos pelo cliente.
